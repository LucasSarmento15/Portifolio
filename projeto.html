Sistema de Gestão de Inventário

Descrição: Criei um sistema de back-end para gerenciamento de inventário, integrando com uma interface front-end em React. O sistema é responsável por processar requisições para atualizar, adicionar ou remover itens de um banco de dados relacional.

Tecnologias: Python (Django), PostgreSQL, Redis, Celery.

Destaques: Otimização de consultas SQL, uso de cache com Redis, e tarefas assíncronas com Celery.

Sistema de Processamento de Dados em Tempo Real

Descrição: Desenvolvi um sistema para processar grandes volumes de dados em tempo real, proveniente de dispositivos IoT, com arquitetura orientada a eventos.

Tecnologias: Python, Apache Kafka, MongoDB, ElasticSearch.

Destaques: Pipeline de dados escalável e processado em tempo real, integração com Apache Kafka para manipulação de fluxos de dados.
